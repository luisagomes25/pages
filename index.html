Sobre o Projeto

O Água em Ação é um sistema de monitoramento inteligente para cisternas e reservatórios, usando sensores e IA para detectar nível da água, pH e rachaduras estruturais. Os dados são enviados por microcontroladores para uma API que os armazena em um banco de dados e disponibiliza em dashboards para tomada de decisão.
