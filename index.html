Sobre o Projeto

O Água em Ação é uma solução tecnológica voltada para o monitoramento automatizado de cisternas e reservatórios, especialmente em comunidades rurais. O sistema utiliza sensores para medir o nível da água e o pH, além de empregar inteligência artificial para detectar possíveis rachaduras estruturais. As informações captadas são enviadas por microcontroladores a uma API, que armazena os dados em um banco de dados e os disponibiliza por meio de dashboards interativos, facilitando a gestão e a tomada de decisão.

Tecnologias Utilizadas

  • Microcontroladores: ESP32, Arduino Uno e Raspberry Pi Pico
  • Sensores de nível de água (boia) e pH
  • Desenvolvimento backend com Python (Flask)
  • Interface web com HTML, CSS e JavaScript
  • Banco de dados MySQL
  • Sistema de autenticação com Flask-Login
  • Hospedagem em nuvem via AWS EC2
  • Inteligência artificial com Hugging Face
  • Comunicação via protocolo TCP
