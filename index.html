Instalação Simplificada

Configure seus sensores em poucos passos. Nossa interface é intuitiva e o processo é otimizado para sua comodidade.

Equipe

  • Gabriel Alves: Implementação da segurança na navegação do sistema, hospedagem em nuvem (AWS EC2) e melhorias nos dashboards para usuários e administradores.
  • João Victor: Correção de bugs e inclusão de novas funcionalidades no frontend para aprimorar a experiência do usuário.
  • Luís Felipe de Arruda Gomes: Desenvolvimento da API e do servidor responsáveis pela comunicação com o ESP32 e Arduino, integrando sensores e banco de dados.
  • Pedro Ivo Cavalcante: Treinamento e implementação da IA com foco na detecção de rachaduras estruturais, utilizando a plataforma Hugging Face.
  • Pedro Marques: Elaboração do edital do projeto, desenvolvimento do layout das interfaces e melhorias visuais no sistema.
  • Vinícius Rodrigues: Implementação da comunicação cliente-servidor via protocolo TCP e integração de sensores ao Raspberry Pi Pico.

Tecnologias Utilizadas

  • Microcontroladores: ESP32, Arduino Uno e Raspberry Pi Pico
  • Sensores de nível de água (boia) e pH
  • Desenvolvimento backend com Python (Flask)
  • Interface web com HTML, CSS e JavaScript
  • Banco de dados MySQL
  • Sistema de autenticação com Flask-Login
  • Hospedagem em nuvem via AWS EC2
  • Inteligência artificial com Hugging Face
  • Comunicação via protocolo TCP
