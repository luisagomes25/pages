O que é o Sensorium?

O Sensorium é uma plataforma completa para monitoramento inteligente de ambientes, integrando sensores físicos (ESP32, Arduino, Raspberry Pi Pico) a um backend robusto em Python (FastAPI), com visualização de dados em dashboards web modernos. Ideal para aplicações acadêmicas, industriais e residenciais.

FastAPI MySQL Docker Pytest Arduino/ESP32 Raspberry Pi Pico HTML5 CSS3 JavaScript AWS EC2

Principais Funcionalidades

Instalação e Deploy Simplificados

Utilize Docker para subir todo o ambiente (backend, banco de dados, testes) com um único comando.

API RESTful Moderna

Backend em FastAPI, com endpoints seguros, JWT, integração com sensores e documentação automática (Swagger/OpenAPI).

Dashboards Interativos

Visualize leituras em tempo real, relatórios e status dos sensores em uma interface web responsiva.

Testes Automatizados

Testes unitários e de integração com Pytest, garantindo qualidade e confiabilidade do sistema.

Integração com Hardware

Comunicação direta com ESP32, Arduino e Raspberry Pi Pico para coleta e envio de dados.

Tecnologias Utilizadas

Análise completa do projeto:

Backend

  • Linguagem: Python
  • Framework Principal: FastAPI (para construção de APIs de alta performance)
  • Validação de Dados: Pydantic (para validação, serialização e documentação automática)
  • ORM: SQLAlchemy (comunicação com MySQL)
  • Autenticação e Segurança:
    • JWT (tokens de acesso para proteger rotas)
    • OAuth2 (fluxo de autenticação, incluindo Login com Google)
  • Servidor ASGI: Uvicorn (execução de alta performance da aplicação)

Frontend

  • Estrutura e Estilo: HTML5, CSS3, Bootstrap 5
  • Interatividade e Visualização de Dados: JavaScript, Chart.js
  • Renderização no Servidor: Jinja2

Banco de Dados

  • Principal: MySQL 8.0
  • Testes: SQLite (em memória para testes unitários)

Testes e Qualidade de Código

  • Pytest (automação de testes unitários e de integração)
  • Coverage.py (análise de cobertura de código)

DevOps e Infraestrutura

  • Docker e Docker Compose (containerização da aplicação e dependências)

Hardware e Comunicação

  • Microcontroladores: ESP32, Arduino, Raspberry Pi Pico W
  • Sensores: Nível de água, pH, luz, umidade, entre outros
  • Protocolos: HTTP/HTTPS (comunicação web e hardware), Serial (Arduino)
